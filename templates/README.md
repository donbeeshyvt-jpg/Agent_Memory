# Template Vaults

This folder stores in-project template vaults for second-brain bootstrap.

- `persona_factory_vault/`: baseline multi-persona template for `brain-seed-template`.

Legacy note:
- `_tmp_persona_factory_vault` and `_tmp_persona_bundle_target` under the workspace root are old test fixtures.
- Core startup no longer depends on those root-level `_tmp_*` folders.
